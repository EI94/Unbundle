Unbundle + Claude

Segnalare use case AI e best practice direttamente da Claude

L'obiettivo è semplice: un collega scrive a Claude in linguaggio naturale, Claude raccoglie le informazioni mancanti e salva il contributo nel workspace Unbundle corretto solo dopo conferma.

La risposta breve

Sì, centralizzabile

La Skill si configura da Admin Claude

Nei piani Claude Team ed Enterprise, un Owner può caricare una Skill a livello organizzazione da Organization settings → Skills. La Skill appare poi agli utenti in Customize → Skills.

Serve anche il connector

La Skill spiega, il connector agisce

La Skill dice a Claude come fare le domande e quando inviare. Per scrivere davvero in Unbundle serve anche un connector/MCP collegato a Unbundle. Nei piani Team ed Enterprise l'Owner lo aggiunge da Organization settings → Connectors.

In pratica: Skill = processo. Connector = accesso sicuro a Unbundle. Per un'esperienza davvero seamless servono entrambi.

Step-by-step per l'Admin Claude

1. Attivare le capacità necessarie

  1. Aprire Claude come Owner o Primary Owner.
  2. Andare in Organization settings → Capabilities.
  3. Attivare Code execution and file creation.
  4. Andare in Organization settings → Skills.
  5. Attivare Skills.

2. Caricare la Skill Unbundle

  1. Preparare un file .zip con la cartella della Skill.
  2. La cartella deve contenere almeno SKILL.md.
  3. In Claude: Organization settings → Skills → + Add.
  4. Caricare lo zip della Skill Unbundle.
  5. La Skill diventa disponibile agli utenti dell'organizzazione.

3. Aggiungere il connector Unbundle

  1. Andare in Organization settings → Connectors.
  2. Cliccare Add.
  3. Scegliere Custom → Web.
  4. Inserire l'URL del connector Unbundle.
  5. Se previsto, inserire OAuth Client ID e Secret.

4. Cosa fa l'utente finale

  1. Va in Claude → Customize → Connectors.
  2. Clicca Connect su Unbundle.
  3. In chat scrive in modo naturale cosa vuole segnalare.
  4. Claude raccoglie i campi mancanti.
  5. L'invio avviene solo dopo conferma esplicita.
Esempio prompt utente: "Claude, voglio segnalare a Unbundle un use case AI: automatizzare il tracking dei progetti. Guidami tu con le domande mancanti."

Il contributo entra in Raccolta & Ranking nel workspace corretto, pronto per review e prioritizzazione.

Risultato desiderato: nessun collega deve conoscere token, API o dettagli tecnici. Deve solo parlare con Claude.

Nota importante sullo stato implementativo

La Skill centralizzata è supportata da Claude Team/Enterprise. Per rendere l'esperienza completamente seamless anche su Claude.ai/Cowork, Unbundle deve esporre un connector remoto MCP raggiungibile via internet, idealmente con OAuth per utente. Il setup locale tramite Claude Code resta utile per pilot tecnici, ma il percorso cliente consigliato è: Skill organizzativa + connector remoto + autorizzazione per utente.

Fonti ufficiali consultate: Claude Help Center, “Provision and manage Skills for your organization”; Claude Help Center, “Get started with custom connectors using remote MCP”; Claude Docs, “Skills overview” e “Creating custom skills”.
